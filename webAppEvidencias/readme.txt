En este proyecto hay una muestra de programas que hago de la asignatura de Desarrollo Web en Entorno Cliente (DWEC) 
del ciclo formatico Desarrollo de Aplicaciones Web. 

Por orden encontramos:
  - Mapa mental del primer proyecto, Recomerçem, enfocado a DWEC. 
  - Mapa mental del primer proyecto, Recomerçem, enfocado a Desarrollo de Interfaces Web (DIW). 
  - Ejercicio en javascript con JSFiddle utilizando estructuras simples. 
  - Ejercicio en javascript con JSFiddle utilizando estructuras compuestas. 
  - Ejercicio en javascript con JSFiddle utilizando callbacks. 
  - Ejercicio en javascript con JSFIddle utilizando métodos de objetos.
